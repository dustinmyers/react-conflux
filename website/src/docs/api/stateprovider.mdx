## Introduction

The `StateProvider` component is the key to `React-Conflux`. This component allows all children components in the descending component tree to receive any of its associate state.

## Import Statement

You can import the `StateProvder` component with the following line:

<br />

```jsx
import { StateProvider } from 'react-conflux';
```

<br />

## Parameters

The `StateProvider` component receives two props arguments, `reducer` and `stateContext`.
